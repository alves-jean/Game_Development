# ET Explorer

## Descrição do Jogo

ET Explorer é um jogo de plataforma 2D desenvolvido na Unity utilizando a linguagem C#. O jogo possui progressão de dificuldade em três níveis, onde o jogador aprende novas mecânicas ao longo da gameplay. O objetivo principal é controlar um explorador espacial, superar obstáculos, derrotar inimigos, coletar diamantes para aumentar a pontuação e alcançar o objetivo final de cada fase.

O jogo foi desenvolvido com foco em movimentação, precisão nos pulos, exploração e progressão de habilidades, proporcionando uma experiência dinâmica e desafiadora.

---

# Mecânicas do Jogo

## Level 1

No primeiro nível, o jogador aprende as mecânicas básicas do jogo:

* Correr para direita e esquerda;
* Pular sobre plataformas;
* Fugir dos enemies;
* Derrotar enemies pulando sobre a cabeça deles;
* Coletar diamantes para aumentar a pontuação.

Ao chegar na bandeira ao final da fase, o player é transportado para o próximo nível.

---

## Level 2

No segundo nível, o jogador utiliza os conhecimentos adquiridos anteriormente e aprende uma nova mecânica:

* Escalar correntes suspensas;
* Pular entre correntes;
* Evitar cair na ZoneDeath;
* Fugir dos enemies;
* Evitar spikes;
* Desviar do peso que cai do teto;
* Continuar coletando diamantes.

Ao alcançar a bandeira, o jogador é transportado para o terceiro nível.

---

## Level 3

O terceiro nível reúne todas as mecânicas aprendidas durante o jogo:

* Pular plataformas com maior precisão;
* Escalar correntes suspensas;
* Fugir de enemies terrestres;
* Evitar enemies flutuantes;
* Desviar de spikes e pesos;
* Coletar diamantes para aumentar a pontuação.

O objetivo final do jogador é alcançar o foguete para concluir o jogo.

---

# Controles

| Ação                | Tecla                   |
| ------------------- | ----------------------- |
| Mover para esquerda | Seta Esquerda           |
| Mover para direita  | Seta Direita            |
| Pular               | Seta Para Cima ou Space |

---

# Sistema de Vidas

O jogador possui um sistema de vidas representado por corações na HUD. Ao sofrer dano por enemies, spikes, quedas ou obstáculos, o player perde uma vida.

---

# Sistema de Pontuação

Durante as fases, o jogador pode coletar diamantes espalhados pelo cenário. Cada diamante aumenta a pontuação total do jogador.

# Instruções de Execução

## Requisitos

* Unity Hub instalado;
* Unity versão compatível com o projeto.

## Como Executar

1. Faça o download ou clone o projeto;
2. Abra o Unity Hub;
3. Clique em “Open Project”;
4. Selecione a pasta do projeto ET Explorer;
5. Aguarde o carregamento dos assets;
6. Abra a cena inicial do jogo;
7. Clique no botão “Play” dentro da Unity para executar o jogo.

---

# Objetivo Final

Complete os três níveis, sobreviva aos obstáculos, derrote os enemies, colete diamantes e alcance o foguete para finalizar a aventura do ET Explorer.
